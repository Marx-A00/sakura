# Milestone v1.0: MVP

**Status:** SHIPPED 2026-04-13
**Phases:** 1-7
**Total Plans:** 17

## Overview

Sakura is a personal Android app (Kotlin + Jetpack Compose) for logging food and workouts to org-mode plain text files that sync to an Emacs/org-roam setup via Syncthing. The roadmap delivers in four phases: a validated org file engine and sync foundation, full food tracking end-to-end, full workout tracking end-to-end, and a unified dashboard with analytics. Every data feature depends on the org engine being correct from day one — that is the singular risk this ordering protects against.

- [x] 01-02-PLAN.md — OrgWriter + OrgParser TDD with Emacs org-lint validation checkpoint
- [x] 01-03-PLAN.md — SyncBackend, SyncthingFileBackend, DataStore, onboarding flow, and device deploy

  1. App installs via ADB on Galaxy S21 FE and launches without crashing

## Milestone Summary

**Key Decisions:**
- Android over iOS (free device, no dev fee, full filesystem access)
- Kotlin + Jetpack Compose (standard modern Android stack)
- Custom OrgEngine (no maintained JVM org library exists)
- No Room database — org files are the sole data store
- MANAGE_EXTERNAL_STORAGE for v1 (sideload-only, no Play Store policy)
- Property drawers as canonical org format for structured data
- Navigation Compose 2 (not Nav 3) — small app doesn't justify Nav 3 API surface
- Dual storage mode: Syncthing power user + zero-config local storage

**Issues Resolved:**
- templateName null after cold start (Phase 6 — derive from splitDay at parse time)
- isPr badge not persisting across restarts (Phase 6 — confirmed round-trip correct)
- LOCAL->SYNCTHING migration data loss (Phase 6 — copy files before switching)
- Tech debt cleanup: dead code removal, fragile error string matching, placeholder notification icon

**Technical Debt Carried Forward:**
- SyncBackendError.ConflictDetected defined but unused (write guard deferred — blocking writes on conflict is bad gym UX)
- SYNC-03 conflict handling is read-only (dashboard badge warns, no per-write guard)
